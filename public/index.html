About the tool

The Mobile Checker is a tool for Web developers who want to make their Web page or Web app work better on mobile devices. Learn more.

Contribute

Looking to contribute to Mobile Checker? There is many ways to get involved in its development. Please take few minutes to read this document to understand how you can help in Mobile Checker development.

Give feedback

There are many ways to send feedback or discuss the new Mobile Checker. You can open an Issue on Github or send an email to our mailing list.
